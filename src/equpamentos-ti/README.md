# Sistema de Cadastro de Equipamentos de TI

Aplicação web desenvolvida em Flask para gerenciamento de equipamentos de tecnologia da informação.

## Funcionalidades

* Cadastro de usuários
* Login
* Logout
* Cadastro de equipamentos
* Listagem de equipamentos
* Atualização de equipamentos
* Exclusão de equipamentos

## Tecnologias

* Python
* Flask
* SQLite
* HTML
* CSS

## Metodologia

* Kanban
